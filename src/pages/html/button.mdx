## Bad Example

```tsx

```

## Accessibility Notes

- Use native button elements
- Don't use div + onClick

## Keyboard Support

| Key   | Action       |
| ----- | ------------ |
| Tab   | Focus button |
| Enter | Activate     |
| Space | Activate     |
